Auditoría Wi-Fi con Aircrack-ng

Guía técnica avanzada, Ago 12, 2025

Aircrack-ng es una suite profesional de auditoría de redes Wi-Fi que permite evaluar la seguridad de redes inalámbricas bajo los estándares IEEE 802.11. Está pensada para expertos en ciberseguridad, administradores de red y entusiastas que desean probar la robustez de sus propias redes.

La suite se compone de varias herramientas clave:

  • airmon-ng: Habilita el modo monitor y gestiona procesos que interfieren con la captura de paquetes.
  • airodump-ng: Captura tramas de gestión, control y datos, registrando BSSIDs, ESSIDs, clientes y canales activos.
  • aireplay-ng: Permite inyectar tráfico en la red para generar handshakes o probar autenticaciones.
  • aircrack-ng: Analiza capturas (.cap) y prueba contraseñas mediante ataques de diccionario o fuerza bruta.

Es importante entender que Aircrack-ng no “hackea” redes de forma automática: requiere capturar un handshake WPA/WPA2, que es un intercambio criptográfico entre un cliente y un punto de acceso durante el proceso de autenticación.

Ejemplo seguro de uso

Auditoría paso a paso

A continuación, un ejemplo práctico para auditar tu red doméstica. Usar únicamente en redes propias o con autorización explícita.

# 1. Colocar la interfaz en modo monitor sudo airmon-ng check kill sudo airmon-ng start wlan0 # 2. Capturar tráfico y localizar tu red sudo airodump-ng wlan0mon # 3. Iniciar captura filtrada en tu red sudo airodump-ng --bssid AA:BB:CC:DD:EE:FF --channel 6 \ --write captura wlan0mon # 4. (Opcional) Forzar reautenticación para obtener el handshake sudo aireplay-ng --deauth 10 -a AA:BB:CC:DD:EE:FF wlan0mon # 5. Probar contraseñas con un diccionario aircrack-ng captura-01.cap -w diccionario.txt # 6. Restaurar la interfaz a modo normal sudo airmon-ng stop wlan0mon sudo service NetworkManager restart

Explicación técnica: El handshake WPA2 es un intercambio de cuatro mensajes EAPOL donde el cliente y el AP negocian la clave de sesión. Aircrack-ng utiliza el handshake capturado y genera PMKs (Pairwise Master Keys) a partir de contraseñas del diccionario, comparándolas con el hash derivado del handshake.

Buenas prácticas y defensa

Endureciendo tu red contra ataques
Esquema de seguridad

Para protegerte de ataques realizados con herramientas como Aircrack-ng, sigue estas recomendaciones:

  • Utiliza WPA3 con SAE siempre que sea posible.
  • Configura contraseñas largas (mínimo 16 caracteres) y aleatorias.
  • Desactiva WPS (Wi-Fi Protected Setup), ya que es vulnerable a ataques offline.
  • Actualiza el firmware de tu router para corregir vulnerabilidades conocidas.
  • Habilita filtrado por MAC solo como medida complementaria, no como única defensa.

Recuerda: la seguridad de la red depende tanto de la robustez de la contraseña como de la configuración general del punto de acceso.
